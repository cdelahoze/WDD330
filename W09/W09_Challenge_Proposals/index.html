Site Name

Stratified sampling

Site Purpose

The purpose of the website is to provide a tool for the design of sampling plans using the stratified probability sampling technique. Data sources are entered via an Excel file that will be converted to local JSON to feed the data source required to create the sampling plan. Subsequently, the sampling plan will be exported in an Excel file. External APIs such as Canvas will be used to create graphics. Some libraries like ml.js for statistical development. Similarly, local storage will be used to facilitate the provisional storage of information..
